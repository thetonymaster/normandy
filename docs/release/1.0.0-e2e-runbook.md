# 1.0.0 E2E Bug-Hunt Runbook

One-time, time-boxed pass. Goal: find major bugs before cutting 1.0.0. Most steps
are **$0 (stubbed)**; only the `verify/*.exs` smokes spend (≤ 15 live Haiku calls total).

## 0. Prereqs
- Docker running. `docker compose -f docker-compose.verify.yml up -d` (Task 1).
- `export API_KEY=<anthropic key>` (only for step 5).

## 1. Floor (gate — $0)
```bash
mix format --check-formatted
mix compile --warnings-as-errors
mix test 2>&1 | tee evidence/01-units.txt
```
Look for: 0 failures. **If red, STOP** — fix the floor before hunting.

## 2. Real-backend stores ($0)
```bash
mix test.postgres 2>&1 | tee evidence/02-postgres.txt
mix test.redis 2>&1 | tee evidence/03-redis.txt
mix test --include distributed --include postgres --include redis 2>&1 | tee evidence/04-distributed.txt
```
Look for: 0 failures. Every failure → a bug-log entry. Do **not** stop on first failure.

## 3. New gap tests ($0)
```bash
mix test.postgres test/agents/turn/server_postgres_e2e_test.exs 2>&1 | tee evidence/05-server-pg.txt
mix test test/behaviours/session_store/concurrent_append_mnesia_test.exs 2>&1 | tee evidence/06-mnesia.txt
mix test.redis test/behaviours/session_store/concurrent_append_redis_test.exs 2>&1 | tee evidence/07-redis-concurrent.txt
```
Look for: PASS = no bug. FAIL = bug found → bug-log entry.

## 4. Stubbed-only dry run of live smokes (free iteration)
```bash
NORMANDY_SMOKE_STUB=true MIX_ENV=test mix run verify/guardrails_live.exs
NORMANDY_SMOKE_STUB=true MIX_ENV=test mix run verify/durable_server_live.exs
```
Look for: scripts run clean against the stub before spending a cent.

## 5. Live smokes (PAID — ≤ 15 Haiku calls total)
```bash
MIX_ENV=test mix run verify/guardrails_live.exs 2>&1 | tee evidence/08-guardrails-live.txt
MIX_ENV=test mix run verify/durable_server_live.exs 2>&1 | tee evidence/09-durable-live.txt
MIX_ENV=test mix run verify/json_smoke_live.exs 2>&1 | tee evidence/10-json-smoke-live.txt
```
Look for: each script prints `LIVE CALLS USED: N` at the end; total across all three must be ≤ 15. The JSON smoke adds ≤ 4 Haiku calls. Any invariant break exits non-zero and prints `INVARIANT FAILED`.

## 6. Triage
Fill `docs/release/1.0.0-bug-log.md`: severity per finding, then the go/no-go table. Release is **go** iff zero open Blockers.

## 7. Teardown
```bash
docker compose -f docker-compose.verify.yml down
```
